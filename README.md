# Code Checker

> An automated tool that scans a code of file and returns useful information

## Prerequisites

* Java SDK
* Jenkins
* ngrok

## What Does It Do?
It returns a the following:

* Total # of lines
* Total # of comments
* Total # of single-line comments
* Total # of block comments
* Total # of lines of comment within block comments
* Total # of TODOs
